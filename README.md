# GalaxyBreyne

**AI Intelligence Briefings & Creative Pipelines**

Live at [galaxybreyne.com](https://www.galaxybreyne.com)

## What This Is

Business website for GalaxyBreyne — an AI-powered intelligence briefing service and creative production studio. Built from scratch with vanilla HTML/CSS/JS and deployed on Vercel.

## Features

- Responsive design with Three.js hero animation
- SEO-optimized with JSON-LD structured data (Organization, Person, WebSite schemas)
- Open Graph and Twitter Card meta tags for social sharing
- `llms.txt` for AI crawler accessibility
- XML sitemap and robots.txt for search engine indexing
- Vercel deployment with custom domain routing

## Tech Stack

| Component | Technology |
|-----------|------------|
| Frontend | HTML5, CSS3, Vanilla JavaScript |
| 3D Graphics | Three.js |
| Fonts | Google Fonts (Orbitron, Inter) |
| Hosting | Vercel |
| SEO | JSON-LD, Open Graph, sitemap.xml |
